# minifoot.

Manager de futebol minimalista em React, TypeScript e Tauri.

Minifoot. Tudo o que importa, nada que atrapalha.

## Scripts

- `pnpm dev`: abre a janela Tauri em modo desenvolvimento.
- `pnpm web:dev`: abre apenas o Vite.
- `pnpm test`: roda os testes unitarios.
- `pnpm build`: valida TypeScript e gera build web.
- `pnpm lint`: roda Biome.
